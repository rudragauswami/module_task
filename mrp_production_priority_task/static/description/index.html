Custom MRP Priority Stars

Extend Manufacturing Order Priority to 3 Stars with Colors

This module enhances the default Odoo Manufacturing Order (MRP) priority functionality. Instead of the standard priority, it extends it to a 3-star rating system with beautiful color decorations, making it easier for your production team to identify urgent orders at a glance.

🌟 Key Features 🌟

  • 3-Star Priority System: Upgrades the standard priority field to support up to 3 stars.
  • Color Decorations: Visually distinct colors for different priority levels on the tree/list view.
  • Seamless Integration: Works perfectly with the standard Odoo MRP and Stock modules.
  • Easy to Use: No complex configuration required. Plug and play!
